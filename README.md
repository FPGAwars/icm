# Icestudio collections manager

Tools for managing [Icestudio](https://github.com/FPGAwars) collections.

## Install

```bash
pip install icm
```

## Commands

|  Command   | Description |
|:----------:|:-----------:|
| [create]() | Create a collection structure. |
| [update]() | Update docs and translation. |
| [validate]() | Validate a collection. |
